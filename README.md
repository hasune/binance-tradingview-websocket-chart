바이낸스로 받은 웹 소켓 데이터를 통해서 실시간으로 트레이딩뷰 차트를 그립니다.  

트레이딩뷰에서 지원하는 lightweight 라이브러리를 이용하면 여러가지 스타일의 차트를 자신이 커스터마이징 하여 표현할 수 있습니다.
